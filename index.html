ap_channel_allocation.py

I've developed this script which helps to optimize the random channel allocation happening in access points. This script also contains code for visualizing the channel allocated to different access points. For seperate code for visualization please see net_vis.py I've optimized channel for minimizing the interference between access points at the same time using as minimum channels for allocation as possible.
